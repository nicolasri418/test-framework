# Propel Projects Test Automation

This repository contains the Playwright test automation framework for the Propel Projects web application. It includes BDD-style tests, page-object patterns, and configurations for Playwright with TypeScript.

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```

2. Run tests
   ```bash
   npm test
   ```

## Structure

- **tests/**: Contains all the test specifications.
- **page-objects/**: Contains the page object model implementations.
- **fixtures/**: Contains test data for data-driven testing.
